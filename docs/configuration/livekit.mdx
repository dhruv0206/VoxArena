# LiveKit Setup

LiveKit handles real-time audio between the browser, voice agent, and phone calls.

## Setup Steps

1. Create an account at [LiveKit Cloud](https://cloud.livekit.io)
2. Create a new project
3. Copy your **API Key**, **API Secret**, and **WebSocket URL**
4. Add credentials to all three `.env` files:

```bash
# backend/.env, frontend/.env.local, and agent/.env
LIVEKIT_API_KEY=your_api_key
LIVEKIT_API_SECRET=your_api_secret
LIVEKIT_URL=wss://your-project.livekit.cloud
```

<Note>
The same LiveKit credentials must match across all three services. Mismatched credentials are the most common cause of agent connection issues.
</Note>

## How It Works

- **Browser sessions**: The frontend generates a LiveKit access token and connects via WebRTC
- **Agent dispatch**: When a room is created, LiveKit dispatches the VoxArena agent worker
- **Phone calls**: Twilio SIP trunks route calls through LiveKit's SIP bridge
